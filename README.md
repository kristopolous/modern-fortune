So fortune is kinda... offensive. 

It is full of outdated jokes that are racially insensitive and punch down to minorities and women.

For instance, in 2017, FreeBSD removed fortune(1) citing it as containing too many offensive parts.

I've spent quite a bit of time [over here](https://github.com/kristopolous/cs-button-announcer) cleaning up the databases and I've extracted them from that project and placed them here.

I've also included a replacement for cowsay called animal-say which uses fortune for the animals. Give it a go!
